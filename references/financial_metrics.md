# Financial Metrics

> **Role in this skill**: Appendix — definitions of common financial metrics. Use as a lookup when you encounter an unfamiliar metric, not as a reading assignment. For how to integrate metrics into a causal thesis, see `references/thesis_first.md` and `references/deep_analysis.md`.

## Sharpe Ratio
Risk-adjusted return: (portfolio return - risk-free rate) / portfolio volatility
- Annualized: Sharpe * √252 (daily returns)
- Higher is better

## Sortino Ratio
Like Sharpe, but only penalizes downside volatility
- Higher is better

## Max Drawdown
Maximum peak-to-trough decline in portfolio value
- Lower is better

## Alpha
Excess return vs benchmark (e.g., S&P 500)
- Positive = outperformance

## Beta
Sensitivity to market movements
- Beta > 1 = more volatile than market
