# Financial Metrics

## Sharpe Ratio
Risk-adjusted return: (portfolio return - risk-free rate) / portfolio volatility
- Annualized: Sharpe * √252 (daily returns)
- Higher is better

## Sortino Ratio
Like Sharpe, but only penalizes downside volatility
- Higher is better

## Max Drawdown
Maximum peak-to-trough decline in portfolio value
- Lower is better

## Alpha
Excess return vs benchmark (e.g., S&P 500)
- Positive = outperformance

## Beta
Sensitivity to market movements
- Beta > 1 = more volatile than market
